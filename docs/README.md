# Documentation

This repo has accumulated a lot of generated windows, reruns, and historical
experiments. The intent of this folder is to make the current state legible
without rewriting that history.

This is closer to an ADR-style design history than a traditional changelog:

- [GB Curtailment Lineage](gb_curtailment_lineage.md): what was built, why it
  changed, and what superseded it
- [Canonical Artifacts](canonical_artifacts.md): which code paths and generated
  outputs are authoritative today
- [../cleanup/README.md](../cleanup/README.md): how generated outputs are now
  classified for keep, archive, and delete

If you are starting work from scratch, read:

1. [GB Curtailment Lineage](gb_curtailment_lineage.md)
2. [Canonical Artifacts](canonical_artifacts.md)
3. [../cleanup/README.md](../cleanup/README.md)
4. [../README.md](../README.md)
